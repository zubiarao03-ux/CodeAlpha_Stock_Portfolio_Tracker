# 📈 Stock Portfolio Tracker

## Project Description

The Stock Portfolio Tracker is a beginner-friendly Python project developed as part of the **CodeAlpha Python Programming Internship**.

This application allows users to select a stock symbol, enter the quantity of shares they own, and calculate the total investment value using predefined stock prices.

Users also have the option to save their investment report into a text file.

---

## Features

- Displays available stock symbols
- Accepts stock symbol from the user
- Accepts quantity of shares
- Calculates total investment
- Saves report as `portfolio_report.txt`
- Handles invalid stock symbols
- Simple and easy-to-understand Python code

---

## Technologies Used

- Python 3
- Visual Studio Code

---

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:

```bash
python portfolio.py
```

4. Enter a stock symbol.
5. Enter quantity.
6. View the total investment.
7. Save the report if desired.
